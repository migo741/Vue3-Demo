# 快速启动指南

## 🚀 启动项目

```bash
npm run dev
```

## 📋 使用步骤

1. 在浏览器中打开项目（通常是 http://localhost:5173）
2. **允许摄像头权限**（这是必须的！）
3. 将手放在摄像头前
4. 尝试以下手势：

### 手势控制

| 手势        | 特效     | 颜色   |
| ----------- | -------- | ------ |
| ✋ 张开手掌 | 螺旋粒子 | 蓝紫色 |
| ✊ 握拳     | 爆炸粒子 | 红橙色 |
| ✌️ 比V      | 波浪粒子 | 青绿色 |
| 👆 竖起食指 | 漩涡粒子 | 紫粉色 |

## 💡 提示

- 确保光线充足
- 手势保持稳定 1-2 秒
- 摄像头画面在右上角
- 左上角显示当前识别的手势和特效

## 🎨 技术栈

- Vue 3 + TypeScript
- Three.js (3D 渲染)
- MediaPipe Hands (手势识别)
- Vite (构建工具)
