# 手势交互粒子系统使用指南

## 🎯 系统特点

这是一个创新的手势交互粒子系统，粒子会根据你的手势变换成不同的形状和文字，并且可以累积多个图案。

## 🌟 核心功能

### 1. 待命状态

- 当没有识别到手势时，8000个粒子会聚集成一个**蓝白色的球体**
- 球体会有轻微的呼吸效果，等待你的指令

### 2. 手势指令

#### ✋ 张开手掌 → 显示"我"

- **动作**: 五指完全伸展，手掌张开
- **效果**: 粒子重组成金色的"我"字
- **位置**: 根据已有图案数量自动排列

#### 🤏 大拇指食指比爱心 → 显示爱心

- **动作**: 大拇指和食指指尖轻轻接触，形成小爱心
- **效果**: 粒子变成粉红色的爱心图案
- **特点**: 使用心形参数方程生成完美爱心

#### 👉 食指指向前方 → 显示"你"

- **动作**: 只伸出食指，其他手指收拢，食指指向摄像头
- **效果**: 粒子重组成金色的"你"字
- **注意**: 食指要明显指向前方（Z轴方向）

#### ✊ 握拳 → 打碎重置

- **动作**: 五指完全收拢成拳头
- **效果**:
  1. 所有图案瞬间爆炸散开
  2. 粒子向四面八方飞散
  3. 1秒后重新聚合成待命球体
  4. 清空所有历史图案

### 3. 图案累积系统

- 每次做出手势，会在屏幕上**添加**一个新图案
- 图案会自动排列（3列布局）
- 可以连续创建多个图案，例如：
  - 张开手掌 → "我"
  - 比爱心 → ❤️
  - 食指指向前 → "你"
  - 结果：屏幕上同时显示 "我 ❤️ 你"

### 4. 手势稳定机制

- 每个手势需要保持**800毫秒**才会触发
- 防止误触和重复触发
- 左上角会实时显示当前识别的手势

## 📊 界面信息

### 左上角控制面板

- **摄像头状态**: 显示系统运行状态
- **手部检测**: ✅ 已检测 / ❌ 未检测
- **当前手势**: 实时显示识别的手势名称
- **当前特效**: 显示正在执行的效果
- **已创建**: 显示当前屏幕上的图案数量

### 右上角视频窗口

- 显示摄像头画面（镜像）
- 绿色线条：手部骨架
- 红色圆点：21个手部关键点

## 🎨 视觉效果

### 颜色方案

- **待命球体**: 蓝白色，科技感
- **"我"字**: 金黄色，温暖
- **爱心**: 粉红色，浪漫
- **"你"字**: 金黄色，呼应
- **爆炸**: 彩色粒子四散

### 动画效果

- **平滑过渡**: 粒子从一个形态到另一个形态平滑移动
- **呼吸效果**: 待命球体有轻微的缩放动画
- **旋转**: 整体场景缓慢旋转
- **爆炸**: 握拳时粒子高速飞散

## 💡 使用技巧

### 最佳实践

1. **光线充足**: 确保环境光线良好
2. **距离适中**: 手距离摄像头30-60cm
3. **手势清晰**: 动作要明确，不要模棱两可
4. **保持稳定**: 手势保持1秒左右
5. **完整显示**: 整只手都在画面内

### 创意玩法

1. **表白模式**: 依次比出 "我" + "❤️" + "你"
2. **重复图案**: 连续比同一个手势创建多个相同图案
3. **快速切换**: 快速切换不同手势观察过渡效果
4. **爆炸重置**: 随时握拳清空重新开始

## 🔧 故障排除

### 手势识别不准确

- 确保手部完全在画面内
- 检查右上角是否显示绿色骨架
- 手势要标准，参考上面的动作说明

### 粒子不变化

- 检查"当前手势"是否正确识别
- 手势需要保持800ms
- 避免手势之间快速切换

### 性能卡顿

- 关闭其他占用GPU的应用
- 减少创建的图案数量（握拳重置）
- 刷新页面重新开始

## 🎬 演示流程

```
1. 打开页面 → 看到蓝白色球体
2. 张开手掌 → 球体变成"我"字
3. 比爱心 → "我"字旁边出现爱心
4. 食指指向前 → 再出现"你"字
5. 握拳 → 所有图案爆炸，恢复球体
```
