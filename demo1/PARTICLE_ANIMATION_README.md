# 3D 粒子动画 - 手势控制系统

## 功能介绍

这是一个基于 Three.js 和 MediaPipe 的交互式 3D 粒子动画系统，通过摄像头捕捉手部动作来实时切换不同的粒子特效。

## 特效列表

1. **螺旋粒子** (蓝紫色)
   - 手势：✋ 张开手掌（五指伸展）
   - 效果：粒子呈螺旋状旋转运动

2. **爆炸粒子** (红橙色)
   - 手势：✊ 握拳（五指收拢）
   - 效果：粒子从中心向外爆炸扩散

3. **波浪粒子** (青绿色)
   - 手势：✌️ 比V（食指和中指伸展）
   - 效果：粒子呈波浪状起伏运动

4. **漩涡粒子** (紫粉色)
   - 手势：👆 竖起食指（仅食指伸展）
   - 效果：粒子呈漩涡状螺旋下降

## 运行项目

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 使用说明

1. 打开浏览器访问项目地址（通常是 http://localhost:5173）
2. 允许浏览器访问摄像头权限
3. 将手放在摄像头前，做出不同的手势
4. 观察 3D 粒子特效的实时变化

## 技术栈

- **Vue 3** - 前端框架
- **Three.js** - 3D 图形渲染
- **MediaPipe Hands** - 手部追踪和识别
- **TypeScript** - 类型安全
- **Vite** - 构建工具

## 注意事项

- 需要在 HTTPS 环境或 localhost 下运行才能访问摄像头
- 确保光线充足，手部清晰可见
- 手势识别需要一定的稳定性，保持手势 1-2 秒
- 建议使用 Chrome 或 Edge 浏览器以获得最佳性能

## 性能优化

- 粒子数量：5000 个（可在代码中调整 PARTICLE_COUNT）
- 使用 BufferGeometry 提高渲染性能
- 使用 AdditiveBlending 实现发光效果
- 实时更新粒子位置和颜色

## 自定义开发

```typescript

  break
```

### 添加新手势

在 `detectGesture()` 函数中添加新的手势识别逻辑。

## 故障排除

1. **摄像头无法打开**
   - 检查浏览器权限设置
   - 确保使用 HTTPS 或 localhost

2. **手势识别不准确**
   - 改善光线条件
   - 调整 `minDetectionConfidence` 参数

3. **性能问题**
   - 减少 PARTICLE_COUNT 数量
   - 降低粒子材质的复杂度
